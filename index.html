IDENTIFICADORES

Objetivo:

Entender a diferença entre os identificadores class e id em HTML e CSS, bem como a especificidade e as boas práticas no uso desses elementos.

Para que serve class ?

    A class é um identificador reutilizável. Ela pode ser atribuída a vários elementos HTML para que recebam os mesmos estilos. Você pode aplicar um estilo a todos os elementos com a mesma class no CSS.

Para que serve id ?

    O id é um identificador único, ou seja, cada id deve ser usado em apenas um elemento da página. No CSS, você usa o símbolo # para aplicar estilos a um id .
